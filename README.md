## Descrição do Projeto

Este repositório contém um **site completo** desenvolvido como projeto de revisão da unidade curricular **"Publicar Aplicações Web"**.  
O projeto reúne, de forma **organizada e prática**, todos os conceitos e tecnologias aprendidos durante o período, demonstrando a **aplicação real dos conhecimentos em desenvolvimento web**.

O site é composto por **6 páginas interconectadas**, que abordam desde os **fundamentos do desenvolvimento web** até a **publicação e hospedagem de aplicações**, criando um **guia completo **.

---

##  Principais Pontos Revisados

### Fundamentos do Desenvolvimento Web
- **HTML5 Semântico:** uso adequado de elementos estruturais (`<header>`, `<section>`, `<article>`, `<footer>`)
- **CSS3 Avançado:** Flexbox, Grid Layout, variáveis CSS, animações e transições
- **Design Responsivo:** media queries e abordagem *mobile-first*
- **JavaScript ES6+:** manipulação do DOM, event listeners e interatividade

---

### Controle de Versão e Colaboração
- **Git:** comandos essenciais (`init`, `add`, `commit`, `push`, `pull`)
- **GitHub:** criação de repositórios, pull requests e trabalho colaborativo
- **Fluxo de Trabalho:** branching, merging e resolução de conflitos

---

### Ambiente de Desenvolvimento
- **Terminal Linux:** comandos essenciais para navegação e manipulação de arquivos
- **Git Bash:** configuração e uso no ambiente Windows
- **Editores de Código:** organização e estruturação de projetos

---

### Publicação e Hospedagem
- **GitHub Pages:** hospedagem integrada com repositórios Git
- **Vercel:** deploy automático e performance otimizada
- **Netlify:** plataforma completa com CI/CD e serverless functions
- **Render:** configuração personalizável para diversos tipos de serviços

---

### Segurança e Infraestrutura Web
- **HTTPS/SSL:** protocolos seguros e certificados digitais
- **DNS:** sistema de nomes de domínio e resolução de endereços
- **Segurança Web:** criptografia, autenticação e proteção de dados

---

## Conclusão

Este projeto demonstra, de forma prática, como planejar, desenvolver, versionar e publicar aplicações web modernas, aplicando boas práticas e tecnologias atuais do mercado.
